# SmartSpend - Expense Tracker

**SmartSpend** is a web-based expense tracker that helps users manage their daily expenses, track budgets, and analyze spending habits. This project demonstrates full-stack development, teamwork, and practical database handling.

---

## Features
- Add, edit, and delete expenses
- Categorize expenses (Food, Travel, Bills, etc.)
- Dashboard overview with total expenses and category-wise breakdown
- User authentication with secure login/signup
- Responsive design for desktop and mobile

---

## Technology Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **Security:** bcryptjs for password hashing, express-session for session management
