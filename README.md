# SmartSpend

**SmartSpend** is a web app to track personal expenses, manage budgets, and visualize spending through an intuitive dashboard.

## Key Features
- User login/signup
- Add, edit, delete expenses
- Categorize expenses (food, travel, bills)
- Dashboard with charts & analytics
- Mobile-friendly design

## Tech Stack
- **Frontend:** HTML, Tailwind CSS, JavaScript  
- **Backend:**  Node.js  ,express.js
- **Database:** MySQL
